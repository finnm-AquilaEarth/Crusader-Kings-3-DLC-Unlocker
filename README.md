# 🎮 CK3Unlocker – Automatic Crusader Kings 3 DLC Activator & Unlocker

**CK3Unlocker** is a lightweight, all-in-one utility designed to automatically unlock and activate downloadable content (DLC) for **Crusader Kings 3** on Steam.  
It provides a fast and safe way to manage all owned expansions and cosmetic packs without manual file edits.

> ⚠️ A legitimate Steam copy of **Crusader Kings 3** is required to unlock DLCs.

# 🧰 Requirements

Before using CK3Unlocker, ensure you have:

* A legitimate **Steam version** of **Crusader Kings 3**  
* Latest release of **CreamAPI**  
* Internet connection (for SteamCMD data fetch)  
* **.NET 7 Runtime** installed  
* Steam running in the background

# ⚡ Installation & Activation Guide

## 1️⃣ Download CreamAPI

Get the newest release of **CreamAPI** and extract it to a dedicated folder (fully portable).

## 2️⃣ Launch CK3Unlocker

Run the executable as **Administrator**.

## 3️⃣ Select Game Directory

Choose your **Crusader Kings 3** installation folder.

## 4️⃣ Activate DLCs

Click **Activate DLCs** and wait for the process to finish.

## 5️⃣ Launch the Game

Start **Crusader Kings 3** via Steam with all DLCs unlocked.

# 🔧 Troubleshooting

| Issue                          | Possible Solution                                                 |
| -------------------------------| ----------------------------------------------------------------- |
| DLCs not appearing in-game      | Run CreamAPI as Administrator and restart Steam                  |
| Game crashes on launch          | Use CreamAPI’s “Uninstall” option and reinstall unlockers        |
| Missing DLL errors              | Reinstall Koaloader via CreamInstaller context menu              |
| Unlocker not working            | Re-run CreamAPI and refresh Steam appinfo for Crusader Kings 3   |
